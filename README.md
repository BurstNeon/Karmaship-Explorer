# Karmaship-Explorer

## Requirements

- nodejs (https://nodejs.org/en/)

## Setup

``` shellsession
npm update | npm install module_name
```

## Configuration
``` yaml
  ## Default HTTP Port
  httpPort: 80
  
  ## Create Log Files Peers
  peersLog: false
  
  ## Display Gather Reward (Frontend)
  gatherReward: 0.15
  
  ## Wallet-URL (internal | external)
  walletUrl: 'http://127.0.0.1'
  
  ## Wallet-Port API CALL
  walletPort: 4930
```
